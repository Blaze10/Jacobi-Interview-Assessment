CUSTOMER Table
Columns: CUSTOMERID (PK), CUSTOMERNAME

SALES Table
Columns: SALESID (PK), CUSTOMERID, AMOUNT

VIP Table
Columns: CUSTOMERID (PK)


Q: What sort of query would I run to create a table showing total sales by customer?

SELECT C.CUSTOMERID, C.CUSTOMERNAME, SUM(S.AMOUNT) AS `TOTAL SALES` FROM CUSTOMER AS C
JOIN SALES AS S ON S.CUSTOMERID = C.CUSTOMERID
GROUP BY C.CUSTOMERID
ORDER BY `TOTAL SALES` DESC;


Q: What sort of query would I runto create a list of customers that are not VIPs?

SELECT C.CUSTOMERID, C.CUSTOMERNAME FROM CUSTOMER AS C
LEFT JOIN VIP AS V ON V.CUSTOMERID = C.CUSTOMERID
WHERE V.CUSTOMERID IS NULL;
